# BlueKyteAI Project

This repository contains the frontend and backend code for the BlueKyteAI Task Management project.

## Prerequisites
Ensure that you have the following prerequisites installed on your system:
- [Git](https://git-scm.com/download/linux)

## Installation & Setup

### 1. Clone the Repository
```bash
git clone git@github.com:jithendra945/bluekyteai_project_1.git
```

2. Backend Setup
```bash
cd backend
```
Follow the instructions in the Backend README for setting up the backend services.

3. Frontend Setup
```bash
cd frontend
```
Follow the instructions in the Frontend README for setting up the frontend services.

By following these steps, you'll have both the backend and frontend components of the BlueKyteAI Task Management project set up on your local machine.

Feel free to reach out if you encounter any issues or have further questions!
